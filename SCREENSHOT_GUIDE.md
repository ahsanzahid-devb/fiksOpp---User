# Screenshot Guide for Google Play Store

## Requirements Summary

### Phone Screenshots
- **Quantity:** 2-8 (minimum 2, recommended 4-8)
- **Format:** PNG or JPEG
- **Size:** Up to 8 MB each
- **Aspect ratio:** 16:9 or 9:16 (portrait recommended)
- **Dimensions:** 320px - 3840px per side
- **Recommended:** 1080 x 1920 pixels (Full HD portrait)

### Tablet Screenshots (Optional)
- **7-inch:** Up to 8 screenshots, same specs as phone
- **10-inch:** Up to 8 screenshots, 1080px - 7680px per side

## How to Take Screenshots

### Method 1: Android Device
1. Open your app on an Android device
2. Navigate to the screen you want
3. Press **Power + Volume Down** simultaneously
4. Screenshot saved to Gallery
5. Transfer to computer for editing

### Method 2: Android Emulator
1. Open Android Studio
2. Run your app on emulator
3. Use emulator's screenshot tool (camera icon)
4. Or use device screenshot (Power + Volume Down in emulator)

### Method 3: Flutter DevTools
1. Run `flutter run` in terminal
2. Open DevTools
3. Use screenshot feature

## Screenshot Editing Tips

### Add Text Labels (Optional but Recommended)
- Add short descriptions above/below screenshots
- Example: "Browse Services" or "Post a Job"
- Use clear, readable fonts
- Keep text minimal

### Add Highlights
- Highlight important features with arrows or circles
- Use subtle colors (don't overwhelm)
- Point to key UI elements

### Framing
- Add phone frame around screenshot (optional)
- Makes it look more professional
- Many tools can do this automatically

### Consistency
- Use same style for all screenshots
- Same text style and positioning
- Same frame style (if using frames)

## Screenshot Dimensions

### Recommended Sizes:

**Portrait (9:16) - Recommended for Phones:**
- **Full HD:** 1080 x 1920 pixels ‚úÖ (Recommended)
- **2K:** 1440 x 2560 pixels
- **4K:** 2160 x 3840 pixels

**Landscape (16:9):**
- **Full HD:** 1920 x 1080 pixels
- **2K:** 2560 x 1440 pixels
- **4K:** 3840 x 2160 pixels

**Note:** Google Play will resize automatically, but 1080x1920 is the sweet spot for quality and file size.

## Upload to Google Play Console

1. Go to **Store presence** ‚Üí **Store listing**
2. Scroll to **Phone screenshots**
3. Click **Upload**
4. Select your screenshot files
5. Drag to reorder (first screenshot is most important)
6. Repeat for tablet screenshots (if applicable)
7. Click **Save**
