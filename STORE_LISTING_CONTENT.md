### 3. Phone Screenshots

**Requirements:**
- **Quantity:** 2-8 screenshots (minimum 2, recommended 4-8)
- **Format:** PNG or JPEG
- **Size:** Up to 8 MB each
- **Aspect ratio:** 16:9 or 9:16 (portrait recommended for phones)
- **Dimensions:** Each side between 320px and 3840px
- **Recommended:** At least 4 screenshots at minimum 1080px on each side for promotion eligibility

**Recommended Screenshots (in order):**
1. **Home/Dashboard Screen** - Show main interface
2. **Service Listing Screen** - Show available services
3. **Job Posting Screen** - Show how to post a job
4. **Provider Profile Screen** - Show provider details
5. **Chat Interface** - Show messaging feature
6. **Booking Details** - Show booking management
7. **Payment Screen** - Show payment options
8. **Booking History** - Show past bookings

**Screenshot Dimensions (Recommended):**
- **Portrait (9:16):** 1080 x 1920 pixels (Full HD)
- **Landscape (16:9):** 1920 x 1080 pixels

**How to Take Screenshots:**
1. Run your app on an Android device or emulator
2. Navigate to each screen
3. Take screenshot (Power + Volume Down)
4. Edit if needed (add text labels, highlights)
5. Resize to required dimensions
