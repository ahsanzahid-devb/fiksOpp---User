# Release Notes for Google Play Console

## Release Notes Format

Release notes should be **short and concise** (maximum 500 characters). They tell users what's new or changed in this specific version, not a full app description.

## How to Use

1. **Copy one of the options above** (or create your own under 500 characters)
2. **Paste it into Google Play Console** in the Release Notes section
3. **For multiple languages**, add each language in its own tag:
   ```
   <en-GB>
   Your English release notes here
   </en-GB>
   
   <en-US>
   Your US English release notes here
   </en-US>
   ```
